# Audio — Ngen

Sube aquí los archivos de audio en formato **MP3** con exactamente estos nombres:

| Archivo | Cuándo se usa |
|---|---|
| `Flow_State.mp3` | Loop durante el Focus (opción 1) |
| `Flow_State_2.mp3` | Loop durante el Focus (opción 2) |
| `Recovery.mp3` | Loop durante el Descanso |
| `Cue_break.mp3` | Señal al quedar 1 minuto en Focus |
| `Cue_flow.mp3` | Señal al quedar 1 minuto en Descanso |

**Notas:**
- Si un archivo falta, la app usa audio sintético como reemplazo — no rompe nada.
- El nombre del archivo debe ser exacto (mayúsculas y minúsculas importan).
- Para subir: botón **Add file → Upload files** en la parte superior de esta carpeta en GitHub.
