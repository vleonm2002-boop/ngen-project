# Audio — Ngen

Sube aquí los archivos de audio en formato WAV con exactamente estos nombres:

| Archivo | Cuándo se usa |
|---|---|
| `Flow_State.wav` | Loop durante el Flow State |
| `Micro_break.wav` | Loop durante el Micro Break |
| `Recovery.wav` | Loop durante la pantalla de recuperación |
| `Sync_1.wav` | Pantalla de sincronización — opción 1 |
| `Sync_2.wav` | Pantalla de sincronización — opción 2 |
| `Sync_3.wav` | Pantalla de sincronización — opción 3 |
| `Cue_break.wav` | Señal al quedar 1 minuto en Flow State |
| `Cue_flow.wav` | Señal al quedar 1 minuto en Micro Break |
| `cue_to_Flow_state.wav` | Transición hacia Flow State |
| `cue_to_recovery.wav` | Transición hacia Recovery |

**Notas:**
- Si un archivo falta, la app usa audio sintético como reemplazo — no rompe nada.
- El nombre del archivo debe ser exacto (mayúsculas y minúsculas importan).
- Para subir: botón **Add file → Upload files** en la parte superior de esta carpeta en GitHub.
